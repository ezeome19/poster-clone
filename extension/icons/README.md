# Extension Icons

You need to add icon files before publishing the extension:

- `icon16.png` — 16×16px
- `icon48.png` — 48×48px  
- `icon128.png` — 128×128px

You can generate these from your app logo. Suggested tools:
- [favicon.io](https://favicon.io) — Generate from text or image
- [Real Favicon Generator](https://realfavicongenerator.net)

Until you add icons, the extension will still work — Chrome will just show a default grey icon in the toolbar.
